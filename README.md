# 🔍 Vuln-Scanner

A web-based vulnerability scanner built with Laravel.

## 👥 Team
- **Batoul** — Frontend & Authentication — [github.com/batoul612](https://github.com/batoul612)
- **Imane** — Backend & Scan Engine — [github.com/RoachXX](https://github.com/RoachXX)

## 🛠️ Tech Stack
- Laravel (PHP)
- MySQL
- Tailwind CSS
- Vite.js

## ⚙️ Installation

git clone https://github.com/batoul612/Vuln-scanner.git
cd Vuln-scanner
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm run dev
php artisan serve

## 🚀 Features
- User Authentication & Registration
- Vulnerability Scanning Engine
- Admin Dashboard
- Scan History & Reports
